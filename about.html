About Us

Non-Commercial Code Available Software is to make a software grouping in the middle of open source software (OSS) and proprietary software. This software grouping would have similarities between open source software (OSS) and proprietary software. This software grouping had the core of OSS motto, but did not fit the definition of open source software by Open Source Initiative like limiting commercial usage. Non-Commercial Code Available Software organization is founded and currently maintained by JstMStudio.



Definition of Non-Commercial Code Available Software

There are many things that define a software to be Non-Commercial Code Available Software. All of these definition had to be meet in order to use Non-Commercial Code Available Software.

  • Allow free usage for non-commercial use.
  • Distributing commercial copies requires written consent or a commercial license from the project developer.
  • Contributors could contribute to the product and use their change non-commercially.
  • Allow users to customize it however they want.
  • Allow modification of the software for non-commercial use.
